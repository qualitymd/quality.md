# Documentation

User- and contributor-facing documentation for the `qualitymd` project, as an OKF
knowledge bundle. Concepts are organized by the four
[Diátaxis](https://diataxis.fr/) modes — tutorials, how-to guides, reference, and
explanation — with the mode recorded as each concept's `type`. Tutorials and
explanation concepts will be added as they're written.

# Modes

- [guides/](guides/) - how-to guides: task-oriented help for getting things done.
- [reference/](reference/) - reference: factual, lookup-oriented material.

# Bundle

- [schema.md](schema.md) - the concept types used in this bundle.
