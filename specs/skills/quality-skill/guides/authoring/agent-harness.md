# Agent-harness Area authoring guide

Governs [`skills/quality/guides/authoring/agent-harness.md`](../../../../../skills/quality/guides/authoring/agent-harness.md).

The guide **MUST** cover the agent harness as a recurring use-context constituent
and partly normative Area for checked-in steering materials: agent entry points,
guidance files, skills, prompts, and related instructions.

The guide **MUST** preserve the domain-agnostic steering-materials factor-family
prompt, the served-domain guardrail, the dual Area/assessment-reference role, and
the projection boundary against Agent Harnessability and the agent audience.
