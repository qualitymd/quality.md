# /quality skill — reference examples

No active checked-in evaluation example corpus is bundled yet.

The previous legacy example runs were removed from the active skill spec tree so
new `/quality evaluate` work is not taught from old assessment, analysis,
recommendation, `report-summary.md`, or `report.json` artifacts. Future examples
must use the evaluation `data/` routine-output graph,
`data/evaluation-output-result.json`, and deterministic report tree.
