# /quality runtime workflows

Workflow instruction files dispatched by `SKILL.md`.

# Workflows

- [Setup](setup.md) - create or update a useful first `QUALITY.md`.
- [Evaluate](evaluate.md) - evaluate a `QUALITY.md` model and write evaluation artifacts.
- [Review](review.md) - review an evaluation result, the model, or a quality concern.
- [Improve](improve.md) - improve the work or model after confirming focus and mutation surface.
- [Update](update.md) - update the `/quality` skill and `qualitymd` CLI pair.
